# TaskManager

> Репозиторий: [zhandos717/task-manager](https://github.com/zhandos717/task-manager)

Минималистичный таск-менеджер для macOS с Pomodoro-таймером и облачной синхронизацией.

## Возможности

### Задачи
- Создание, редактирование, удаление задач
- Три уровня приоритета: высокий / средний / низкий
- Теги с цветовой маркировкой (6 цветов)
- Подзадачи с прогресс-счётчиком
- Дедлайны с визуальным предупреждением о просрочке
- Повторяющиеся задачи (ежедневно / еженедельно / ежемесячно)
- Фильтрация: все / активные / завершённые
- Поиск по названию и заметкам

### Pomodoro
- Таймер 25 мин работы + 5 мин перерыва (настраивается)
- Запуск на конкретную задачу — счётчик 🍅 на задаче
- Уведомления при смене фаз
- Пауза / продолжение / стоп

### Меню-бар (System Tray)
- Иконка с обратным отсчётом во время работы
- Список активных задач — клик запускает Pomodoro
- Быстрое добавление задачи без открытия окна

### Синхронизация
- REST API синхронизация с `zhandos.top`
- Авторизация по `X-Internal-Key`
- Pull при запуске, push при каждом изменении

### Прочее
- Горячая клавиша `⌘⇧Space` открывает окно из любого приложения
- Экспорт задач в Markdown (`⌘E`)
- Статистика: выполнено сегодня, сколько помодоро
- Приложение остаётся в меню-баре после закрытия окна

## Требования

- macOS 15.2+
- Xcode 16+
- Swift 5.9+

## Сборка

```bash
git clone <repo>
open TaskManager/TaskManager.xcodeproj
```

Нажми `⌘R` в Xcode.

## Структура проекта

```
TaskManager/
├── Task.swift              — модель TodoItem, SubTask, TagColor, Recurrence
├── TaskStore.swift         — хранилище (UserDefaults), статистика
├── APIService.swift        — HTTP клиент (URLSession)
├── SyncService.swift       — оркестрация push/pull синхронизации
├── PomodoroTimer.swift     — таймер с фазами и уведомлениями
├── StatusBarManager.swift  — иконка в меню-баре
├── AppDelegate.swift       — жизненный цикл, горячие клавиши
└── ViewController.swift    — весь UI (sidebar, таблица задач, виджеты)
```

## API

Приложение синхронизируется с REST API сервера [zhandos717/tgfinance](https://github.com/zhandos717/tgfinance). Введи ключ в виджете синхронизации (иконка 🔑 в сайдбаре).

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/tasks` | Получить все задачи |
| POST | `/api/tasks` | Создать задачу |
| PATCH | `/api/tasks/:id` | Обновить задачу |
| PATCH | `/api/tasks/:id/status` | Изменить статус |
| DELETE | `/api/tasks/:id` | Удалить задачу |

Заголовок авторизации: `X-Internal-Key: <ключ>`
